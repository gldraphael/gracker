# Setting up your local development environment

## Quickstart

1. Run PostgreSQL and RabbitMQ with docker compose:

    ```sh
    docker compose pull
    docker compose up -d
    ```
1. This will make the following services available:   
      Port | Servce  
    ------:|---------
     17751 | RabbitMQ
     17752 | RabbitMQ management 
     17753 | PostgreSQL
     17754 | Adminer
